    Document Header File

    QDoc only accepts documentation written files with the extensions defined in sources.fileextensions (usually cpp). Any documentation contained in header files is ignored.

    So our header files remains unchanged:

    namespace MyNamespace
{
  class MyClass
  {
  public:
    int myVar;

    int myFunction() const;
  };
}
